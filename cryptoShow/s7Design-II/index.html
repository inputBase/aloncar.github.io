Kako React 'radi'?

1. Označite komponente na osnovu mockup-a

2. Napravite statičku verziju, bez interakcije

3. Identifikujete minimalnu ali konačnu listu (stanje) 'state'

4. Implementirate event handling, odnosno 'state'

5. Ako je potrebno, onda dodate 'Inverse Data Flow'
